# Release Notes: v0.1.0-rc.1

`v0.1.0-rc.1`은 교사용 대시보드 위젯개발 프로젝트의 1차 릴리즈 후보입니다.

- 태그명: `v0.1.0-rc.1`
- 기준 커밋: `9a551e8cd2c0574f7614c98bbb5ba57f9defc7b8`
- 원격 저장소: `https://github.com/surether/dash-board-teacher.git`
- 상태: 원격 태그 push 완료

이번 릴리즈 후보는 기본 대시보드 위젯, 학생명렬 CSV import, 저장된 학생명렬 기반 수업도구 연결을 포함합니다.

## 2. 포함된 주요 기능

- 날짜/시간 위젯
- 커스텀 알림

- 학생명렬 CSV import
- 학생명렬 저장 및 새로고침 후 유지

- 타이머, 뽀모도로, QR코드
- 바로가기/중요사이트 정적 링크 위젯
- 정적 시간표 위젯
- 정적 학사일정 위젯
- 정적 날씨 · 등교 준비 위젯

## 3. 학생명렬 import 기준

현재 학생명렬 import는 CSV 파일 기준으로 검증되어 있습니다.

지원 CSV 헤더:

```csv
학년,반,번호,성명,비고
```

예시:

```csv
학년,반,번호,성명,비고
1,1,1,김가온,
1,1,2,이나래,
1,1,3,박도윤,
```

동작 기준:

- 정상 학생만 저장합니다.
- 오류 행은 저장 대상에서 제외합니다.
- 저장 후 `StudentRosterWidget`에서 표시됩니다.
- 저장 후 뽑기, 사다리, 룰렛에서 학생명렬을 사용할 수 있습니다.
- 브라우저를 새로고침해도 저장된 학생명렬이 유지됩니다.
- 적용 시 기존 저장 학생명렬은 선택한 CSV의 정상 학생 목록으로 교체됩니다.

## 7. 사용 전 주의사항

- 이 버전은 정식 릴리즈가 아닌 release candidate입니다.
- 실제 학교 현장 사용 전 브라우저, 화면 크기, 데이터 초기화 흐름을 추가 확인해야 합니다.
- `.xlsx`, 성적, 상담 데이터는 아직 대시보드 데이터로 연결하지 않아야 합니다.
- 실시간 날씨나 외부 API가 필요한 기능은 후속 Phase에서 별도로 구현해야 합니다.
- 현재 시간표, 학사일정, 날씨 · 등교 준비 위젯은 정적 예시 위젯입니다.
