# Field Test Checklist: v0.1.0-rc.1

## 1. 목적

이 문서는 교사용 대시보드 `v0.1.0-rc.1`을 학교 현장 또는 교사용 시범 사용 전에 점검하기 위한 체크리스트입니다.

이 문서는 기능 추가 문서가 아니라, 현재 릴리즈 후보를 실제 환경에서 확인하기 위한 검증 기준을 고정합니다.

## 5. 학생명렬 CSV import 점검

기준 CSV 헤더:

```csv
학년,반,번호,성명,비고
```

권장 테스트 CSV:

```csv
학년,반,번호,성명,비고
1,1,1,김가온,
1,1,2,이나래,
1,1,3,박도윤,
1,1,,번호누락,
1,1,5,,성명누락
```

체크 항목:

- [ ] 정상 CSV 선택 시 preview가 표시되는가
- [ ] 학생 후보 미리보기가 표시되는가
- [ ] 정상 학생 수가 올바르게 집계되는가
- [ ] 오류 행 수가 올바르게 집계되는가
- [ ] 번호 누락 행이 오류로 표시되는가
- [ ] 성명 누락 행이 오류로 표시되는가
- [ ] 정상 학생만 저장되는가
- [ ] 저장 후 `StudentRosterWidget`에 반영되는가
- [ ] 새로고침 후 학생명렬이 유지되는가
- [ ] 헤더가 다른 CSV에서 앱이 깨지지 않는가
- [ ] 빈 CSV에서 앱이 깨지지 않는가
- [ ] `.xlsx` 선택 시 unsupported 흐름이 유지되는가

## 6. 학생 기반 수업도구 점검

- [ ] 저장된 학생명렬이 뽑기 위젯에 연결되는가
- [ ] 출석 학생 기준 뽑기가 동작하는가
- [ ] 저장된 학생명렬이 사다리 위젯에 채워지는가
- [ ] 저장된 학생명렬이 룰렛 위젯에 채워지는가
- [ ] 학생명렬이 없을 때 빈 상태 안내가 적절히 표시되는가

## 7. 정적 위젯 점검

- [ ] 바로가기/중요사이트 링크가 표시되는가
- [ ] 링크가 새 탭으로 열리는가
- [ ] 시간표가 월~금, 1~7교시 형태로 표시되는가
- [ ] 시간표가 현재는 기본 예시임을 안내하는가
- [ ] 학사일정이 정적 일정으로 표시되는가
- [ ] 학사일정이 기본 예시임을 안내하는가
- [ ] 날씨 · 등교 준비 위젯이 실시간 날씨가 아님을 안내하는가
- [ ] 외부 API/geolocation 없이 동작하는가

## 8. 데이터 유지 및 초기화 점검

- [ ] 잘못된 데이터가 있어도 화면 전체가 중단되지 않는가
